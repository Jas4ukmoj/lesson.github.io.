Урок 1
  1. Заміна стилю прямо в html документі в середині якогось тегу: (style="color:green")
  2. Ще заміна стилю може писатися в частині head стилем CSS (якщо мало стилів, бо багато краще писати окремим файлом CSS).

  3. Можемо в CSS міняти стилі через атрибути. Можемо в CSS підставити любий атрибут (в,strong,p,хоть який) для зміни стилю.

  4. Присвоєння класу для зміни стилю (class="assigned-class")
  5. "класи" можна багато раз вставляти куди треба. Щоб повторявся стиль там, де вам треба.
  6. Через тег link rel="stylesheet" href="URL"> підключаєм CSS файл.
